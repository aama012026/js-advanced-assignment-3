### DOTA performance view
- Write your 9 digit steam account-id (friend-code) or search for it with your persona-name
- Get a summary of recent matches and detailed performance analysis for selected match
- Request a parse of a match, wait a bit, and try to view the same match. It should contain more information

## Learnings
Working with this in typescript has been a steep learning curve in data-piping and validation. While I will continue to develop this project to a portfolio piece, it is largely showing the data in console logs in its current state. I suggest taking a look at the code after testing to see the brunt of the work.
